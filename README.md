# px

Simple X color picker. Mouse left-click to pick up the color.

## Build

Dependencies:

+ make
+ gcc
+ Xlib

```
$ make
```

## Install

```
# make install
```

## Uninstall

```
# make uninstall
```

## Example

```
$ px -a
```

To see all available options for px, run: `px -h`
